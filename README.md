## Description

A privacy-first Android application designed to help users manage their mental well-being through intelligent journaling. By leveraging the Google Gemini API, MindSync provides personalized reflections and insights based on user entries. This project serves as a practical implementation of on-device data privacy using Room DB and modern UI development with Jetpack Compose.

---

## Key Features

* **AI Reflections:** Personalized insights and mindfulness tips generated by Google Gemini.
* **Privacy-First Storage:** Local-only storage using Room Persistence Library to ensure data never leaves the device.
* **Minimalist UI:** A clean, calming interface designed to reduce cognitive load during reflection.
* **Sentiment Tracking:** Visual feedback on entry patterns to help users recognize emotional trends over time.
